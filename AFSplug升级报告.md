| `stereo-strong` | 突出功能 | 强阴影+发光边框 |

| `stereo-float` | 悬浮卡片 | 浮动+阴影，突出显示 |
| `stereo-peak` | 顶部强调 | 顶部高光，山峰效果 |
| `stereo-wave` | 动态效果 | 底部波动线，动态感 |
| `stereo-hover` | 交互强调 | 悬停时增强立体效果 |
| `stereo-btn` | 所有按钮 | 按压反馈，立体交互 |
| `stereo-tag` | 标签元素 | 轻微立体，层次分明 |

- **优势**: 无损缩放、自定义渐变、动态效果

- **交互**: 悬停、点击、激活状态效果
- **响应式**: 移动端自动调整阴影强度

## 🔧 自定义建议

### 1. **调整立体强度**
```css
/* 在CSS变量中调整 */
--depth-light: rgba(255, 255, 255, 0.15);  /* 提高透明度增强立体 */
--depth-shadow: rgba(0, 0, 0, 0.4);        /* 提高不透明度增强阴影 */
```

### 2. **添加更多效果**
现有的立体效果系统可以轻松扩展：
```css
/* 新增自定义立体效果 */
.my-custom-stereo {
  box-shadow: 
    10px 10px 30px var(--depth-shadow),
    -10px -10px 30px var(--depth-light),
    0 0 40px rgba(0, 229, 255, 0.4);
}
```

### 3. **动画调整**
在`@keyframes`部分可调整动画速度和样式。
