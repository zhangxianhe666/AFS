# AFS 新界面特性 - 科技感增强

已成功升级AFS Web界面，添加了以下未来科技感的设计元素：

## 🚀 核心视觉升级

1. **深空背景** (#05050a)
   - 比原版(#0a0a0f)更深，增强沉浸感

2. **动态粒子背景**
   - Canvas实现的浮动粒子网络
   - 70个发光粒子随机移动
   - 粒子间有连接线形成"神经网络"效果
   - 实时动画，性能优化

3. **毛玻璃效果**
   - 玻璃态卡片：`backdrop-filter: blur(14px) saturate(160%)`
   - 半透明面板：透明度60%
   - 支持Firefox/Chrome/Safari

4. **霓虹光环系统**
   - 开关按钮：选中时发光 `box-shadow: 0 0 12px rgba(0,229,255,0.35)`
   - 状态指示灯：脉波动画 `animation: pulse-glow 2.5s ease-in-out infinite`
   - 悬停效果：卡片边框发光

## 🌈 动画与交互

1. **标题文字呼吸光**
   ```css
   animation: text-shimmer 3s ease-in-out infinite
   text-shadow: 0 0 14px rgba(0,229,255,0.4)
   ```

2. **浮动卡片入场**
   ```css
   animation: float-up 0.5s ease-out
   transform: translateY(6px) → 0
   ```

3. **数据卡片悬停**
   - 上浮3像素 `transform: translateY(-3px)`
   - 边框变亮 `border-color: rgba(0,229,255,0.25)`
   - 投影增强 `box-shadow: 0 6px 24px rgba(0,229,255,0.06)`

4. **按钮动态**
   - 主按钮：悬停上浮1px + 光影扩展
   - 幽灵按钮：悬停边框变蓝

## 🖥️ 界面细节

1. **扫描线叠加层**
   - 模拟CRT显示器效果
   - 2px间隔的重复渐变
   - 超低透明度(0.025)不影响可读性

2. **渐变光晕**
   - 顶部青色径向渐变
   - 底部金色径向渐变
   - 营造立体空间感

3. **终端日志样式**
   - 仿命令行界面：黑色背景+青色边框
   - 内部扫描线叠加
   - 日志分类颜色：错误(红)、警告(橙)

4. **霓虹标签系统**
   ```css
   .tag {
     background: rgba(0,229,255,0.08);
     border: 1px solid rgba(0,229,255,0.15);
     color: var(--accent);
   }
   ```

## 🎨 配色方案

- **主色调**: 青色 `#00E5FF` (霓虹蓝)
- **辅助色**: 金色 `#FFD700` (高亮)
- **成功色**: 亮绿 `#00e676`
- **错误色**: 亮红 `#ff5252`
- **警告色**: 橙色 `#ff9100`
- **文字**: 亮灰 `#c8c8d0`
- **暗文字**: 灰 `#6a6a7a`

## 📱 响应式优化

- 移动端(<768px)：卡片堆叠，统计转为2列
- 标题行自适应换行
- 触摸友好的按钮/开关尺寸

## ⚡ 性能考虑

- 粒子动画使用 `requestAnimationFrame`
- 连接线计算使用距离筛选(130px内)
- CSS动画主要使用 `transform` 和 `opacity` (GPU加速)
- 粒子数量控制在70个，避免性能问题
